## 3.超链接
Markdown支持两种超链接语法：行内式和参考式，行内式居多
### 3.1 行内式

[]里写连接文字，()里写链接地址，()中添加空格后的""为链接指定title属性，也可以不加。  
添加后鼠标在超链接悬停会显示title里的文字  
	1. Bangumi[番组计划](http://bangumi.tv/)
	2. Bangumi[番组计划](http://bangumi.tv/"番组计划")
展示效果
- 1.Bangumi[番组计划](http://bangumi.tv/)
- 2.Bangumi[番组计划](http://bangumi.tv/ "番组计划")

### 3.2 参考式
适合添加链接较多的文章，方便对添加的链接进行管理

  
[链接文字][链接标记]  
	1. [百度][1]
	2. [番组计划][2]
	3. [osu!][3]
	4. [osu!][4]
	4. [推特][]
	2. [1]:www.baidu.com
	3. [2]:bangumi.tv
	4. [3]:osu.ppy.sh
	5. [4]:osu.ppy.sh "osu!"
	5. [推特]：https://twitter.com/

展示效果
- [百度][1]
- [番组计划][2]
- [osu!][3]
- [osu!][4]
- [推特][]
[1]:www.baidu.com
[2]:bangumi.tv
[3]:osu.ppy.sh
[4]:osu.ppy.sh "osu!"
[推特]:https://twitter.com
### 3.3 自动连接
<>之间包裹的链接会自动转换
	1. <osu.ppy.sh>
	2. <asroe@outlook.com>

展示效果  
<http://osu.ppy.sh>
<asroe@outlook.com>
## 4.列表
### 4.1无序列表
使用 * + - 表示无序列表
	1. - 标题
	2. - 标题
	3. - 标题
展示效果
- 标题
- 标题
- 标题  

### 4.2有序列表
数字链接着一个英文句点
	1. 1.标题
	2. 2.标题
	3. 3.标题

展示效果
1. 标题
2. 标题
3. 标题

### 4.3定义形列表
一行写定义，下一行解释。  
解释写法：`：+Tab`
	1. Markdown
	2. :	轻量级文本标记语言
	3. Asroe
	4. :	???

### 4.4列表缩进
列表项目标记通常是放在最左边，但是其实也可以缩进，最多 3 个空格，项目标记后面则一定要接着至少一个空格或制表符。
	*   轻轻的我走了， 正如我轻轻的来； 我轻轻的招手， 作别西天的云彩。
	那河畔的金柳， 是夕阳中的新娘； 波光里的艳影， 在我的心头荡漾。 
	软泥上的青荇， 油油的在水底招摇； 在康河的柔波里， 我甘心做一条水草！ 
	*   那榆荫下的一潭， 不是清泉， 是天上虹； 揉碎在浮藻间， 沉淀着彩虹似的梦。 
	寻梦？撑一支长篙， 向青草更青处漫溯； 满载一船星辉， 在星辉斑斓里放歌。 
	但我不能放歌， 悄悄是别离的笙箫； 夏虫也为我沉默， 沉默是今晚的康桥！ 
	悄悄的我走了， 正如我悄悄的来； 我挥一挥衣袖， 不带走一片云彩。
展示效果
*    轻轻的我走了， 正如我轻轻的来； 我轻轻的招手， 作别西天的云彩。
那河畔的金柳， 是夕阳中的新娘； 波光里的艳影， 在我的心头荡漾。 
软泥上的青荇， 油油的在水底招摇； 在康河的柔波里， 我甘心做一条水草！

寻梦？撑一支长篙， 向青草更青处漫溯； 满载一船星辉， 在星辉斑斓里放歌。

### 4.5包含引用的列表
若在列表内放入引用，前加`>`,需要缩进
	1. Asscention
	2. >Asroe
	3. >Aaxscope
展示效果
* Asscention
	>Asroe
	>Aaxscope

### 4.6包含代码的列表
如果要放代码区块的话，该区块就需要缩进两次，也就是 8 个空格或是 2 个制表符：
- Asroe
		Asscention
- Asscention
		Asroe

### 4.7转义字符
1986. What a great season.  

可在.前添加转义字符
	```1986\. What a great season.```
展示效果
1986\. What a great season.
